Welcome to Sporta

With Sporta you have the possibility to:

Create Athletes

Use tab "Athletes" to both create athletes and display all created athletes. Use the buttons delete respectivley update to change attributes. In addition you have the possibility to view all finished workout of an specific athlete.

Create Workouts

Use tab "Workouts" to both create workouts and display all created workout. Use the buttons delete respectivley update to change attributes.

Enter Workout Data to created workouts and assign athletes

Use tab "Workout Data". Use the buttons delete respectivley update to change attributes.

Analyze all finished workouts wheater from all athletes or an specific athlete

Below tab "Workout Data" choose option "Show Plots and KPI" to analyze the data.
